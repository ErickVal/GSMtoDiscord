# GSMtoDiscord
Send Webhook from GSM to Discord

=== Installation ===

- Install NodeJS
- Run `npm install`

=== Usage ===

On GSM Setting > Discord > Webhook put http://localhost:5001
Print: https://i.imgur.com/wQPB5z5.png

create .env file on root directory and type: 
WEBHOOK = 'YOUR_WEBHOOK_DISCORD'
save file

- Run `node app.js`
